    Properties

    customViews

    Array

    Array of various time-ranges in scheduler. Each time range specifies

    • display : Name of the range
    • code : Short code to identify the range
    • fitWidth : fit width of the scheduler and avoid scrollbar
    • dayWidth : Width for each day
    • daycount : Number of days to show
    • interval : interval of days (break days into )
    • hourInterval : interval in hours
    • showHours : Show hours for a day?
    • timeFormat : Format of time to be displayed on X-AXIS

    Default: 0

    defaultMode

    String

    Code of the itme range to be displayed.

    Default: "C"

    endTime

    Number

    End time of calendar. Number 1 through 24. Number 0 through 24. Number must be greater than startTime Minutes are not supported at this time.

    Default: 0

    options

    Object

    All options for AppointmentManager

    showOnStartup

    Boolean

    Make this the default view to be displayed when calendar loads

    Default: true

    showSubHeader

    Boolean

    Option to hide/display the sub-header

    Default: 0

    slotTemplate

    String

    Template to display the items in Y-Axis. Specify the ID of HTML Element to be used.

    Default: ""

    startTime

    Number

    Start time of calendar. Number 0 through 24. Number must be lesser than endTime Minutes are not supported at this time.

    Default: 0
